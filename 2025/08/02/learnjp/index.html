日本語勉強ノート

語彙・句子解釋：

1-30

  1. パチクリさせる
    　👉 眨巴眼睛（表示驚訝或困惑）
    　👀 例：目をパチクリさせる。

  2. ためらいつつ
    　👉 雖然猶豫、邊猶豫邊做某事
    　📌 「つつ」＝一邊～一邊～

  3. 顔（かお）をふせる
    　👉 低下頭、不敢直視（通常表羞愧、難過）

  4. 差（さ）しだす
    　👉 伸出、遞出（如手或物品）

  5. 俺（おれ）は気（き）を取（と）り直（なお）して口（くち）をはさむ。
    　👉 我振作精神插了句話。

  6. 八奈見（やなみ）さんがフォローらしい発言（はつげん）。
    　👉 八奈見先生說了像是打圓場的話。

  7. メリットがない。
    　👉 沒有好處、沒有優勢。

  8. ニヤリと邪悪（じゃあく）な笑（え）みを浮（う）かべる八奈見（やなみ）。
    　👉 八奈見露出邪惡的笑容。

  9. ドレッサー
    　👉 化妝台（dresser）

  10. 俺（おれ）は櫻井（さくらい）君（くん）がゴボウを削（けず）る手際（てぎわ）を、感心（かんしん）しながら眺（なが）めていた。
    　👉 我一邊佩服地看著櫻井君削牛蒡的手法。

  11. 弁当（べんとう）のおかず
    　👉 便當的配菜

  12. ときおりこうやって場所（ばしょ）を使（つか）わせてもらっているらしい。
    　👉 偶爾好像能借這個地方用。

  13. 落（お）しブタ
    　👉 鍋中加壓時放在食物上面的蓋子（壓蓋）

  14. 気負（きお）う
    　👉 過度用力、太過拼命

  15. このダシ、昆布（こんぶ）と鰹（かつお）の合わせですか？
    　👉 這湯底是昆布和柴魚一起熬的嗎？

  16. 手（て）を回（まわ）す
    　👉 暗中處理、走後門

  17. 櫻井（さくらい）君（くん）はこともなげに言（い）うと
    　👉 櫻井君若無其事地說道

  18. 相手（あいて）のために親身（しんみ）になる子（こ）
    　👉 為對方著想的貼心孩子

  19. キッパリ告（つ）げる
    　👉 斷然地說出來、明確表達

  20. ゴボウのキンピラとインゲンの胡麻（ごま）あえ
    　👉 牛蒡炒絲和四季豆芝麻拌菜

  21. モジモジとうつむいている真ん中（まんなか）の子（こ）
    　👉 扭扭捏捏低著頭的中間那個孩子

  22. 初々（ういうい）しい雰囲気（ふんいき）
    　👉 青澀、純真的氣氛

  23. 綺麗（きれい）にラッピングされたマドレーヌ
    　👉 包裝得很漂亮的瑪德蓮蛋糕

  24. からみやすい
    　👉 容易打交道、好搭話（也可指麵條容易吸附醬汁）

  25. チラリと見（み）る
    　👉 偷看一眼、瞥見

  26. 陽（ひ）がかたむくと肌寒（はだざむ）さを感（かん）じる
    　👉 太陽西斜時會感覺到涼意

  27. 終（お）えた
    　👉 結束了、完成了

  28. 暗（くら）がり
    　👉 黑暗處、陰暗處

  29. 湯気（ゆげ）を立（た）てる鍋（なべ）をのぞきこむ
    　👉 探頭看冒著熱氣的鍋子

  30. タンクトップにショートタイツ姿（すがた）
    　👉 穿著背心和短緊身褲的樣子

    31-60

  31. ヒラヒラと手（て）を振（ふ）って出（で）ていく
    　👉 輕飄飄地揮手離去

  32. 思（おも）わずきょどる
    　👉 不由自主地慌張、不安（俗語：挙動不審）

  33. ケラケラと笑（わら）う
    　👉 咯咯笑（輕快的笑聲）

  34. 俺（おれ）がつられて笑（わら）う
    　👉 我也被帶動地笑了出來

  35. 断（だん）じて違（ちが）い
    　👉 絕對不對、絕不是這樣

  36. 首筋（くびすじ）にまとわりつく
    　👉 纏繞在脖子周圍（如汗、髮絲、氣味等）

  37. ペコリと頭（あたま）を下（さ）げる
    　👉 輕輕地鞠躬

  38. ヒョコリとまわりこんできた
    　👉 突然地繞到旁邊（或背後）出現

  39. 俺（おれ）はモヤモヤと振（ふ）り払（はら）い
    　👉 我把心中的煩悶甩開

  40. 訪（おとず）れる
    　👉 拜訪、造訪（也可以是時間或季節的到來）

  41. 小鞠（こまり）が柵（さく）にもたれながらパターロールを食（た）べていた
    　👉 小鞠靠在欄杆上吃著奶油卷

  42. うっかり
    　👉 不小心、粗心大意

  43. ストロー
    　👉 吸管

  44. 言葉（ことば）に詰（つ）まってカレーパンをかじっている
    　👉 不知道說什麼，咬了一口咖哩麵包

  45. 節操（せっそう）のないやつめ
    　👉 沒原則的人啊！（帶有責怪或戲謔語氣）

  46. ためらいながら口（くち）を開（ひら）く
    　👉 猶豫地開口說話

  47. ボンヤリと遠（とお）くを眺（なが）めた
    　👉 發呆地望向遠方

  48. 市街地（しがいち）
    　👉 城市區、市中心

  49. 上機嫌（じょうきげん）の志喜屋（しきや）さんは、デコった指先（ゆびさき）でチップをつまむ
    　👉 心情很好的志喜屋小姐用裝飾過的指甲夾起了小吃

  1. 俺（おれ）はタイミングを見計（みはか）らい、なにげなさを装（よそお）って口（くち）を開（ひら）く
    　👉 我抓準時機，裝作若無其事地開口說話。

  1. 収（おさ）まる
    　👉 平息、歸位、容納、解決（根據語境可指情緒或局勢安定）

  2. 細（ほそ）い指（ゆび）でシロクマの駒（こま）をもてあそぶ
    　👉 用細長的手指把玩北極熊的棋子。

  3. クマの駒（こま）をつつく
    　👉 戳戳熊的棋子（表現無聊、或調戲）

  4. 俺（おれ）は気（き）もそぞろにサイコロを転（ころ）がしながら、会話（かいわ）を続（つづ）ける
    　👉 我心不在焉地擲骰子，一邊繼續聊天。

  5. しゃがんで…
    　👉 蹲下來……

  6. ウキウキと小（ちい）さく身体（からだ）
    　👉 身體小幅度地興奮跳動（或：情緒輕快）

  7. 気楽（きらく）にブラを外（はず）す間柄（あいだがら）だっけ
    　👉 是能輕鬆脫胸罩的關係嗎（表示關係非常親密）

  8. ふらりと立（た）ち上（あ）がる
    　👉 漫無目的地站起來

  9. 頬（ほお）すりちないでください
    　👉 請不要蹭臉（＝すりつけない的誤變形，較口語）

  10. 羽目（はめ）
    　👉 結果、下場（常用在「～する羽目になる」＝落得～下場）

61-90

  1. プンスカ
    　👉 氣呼呼的樣子（擬態語）

  2. イルミネーションをダシに呼（よ）び出（だ）したのだ
    　👉 借口燈飾（裝飾燈）把你叫出來了

  3. おれは曖昧（あいまい）にごまかす
    　👉 我含糊其辭地搪塞過去

  4. 引（ひ）きつった表情（ひょうじょう）
    　👉 僵硬的表情（因緊張、害怕或勉強微笑）

  5. そんなあっさり諦（あきら）めてくれるんだ
    　👉 竟然這麼乾脆就放棄了？

  6. すまし顔（がお）
    　👉 裝作若無其事、冷靜的表情

  7. ガタガタッ
    　👉 咯噔咯噔聲（擬聲，物體搖晃碰撞聲）

  8. 欲（ほ）しがる人（ひと）と見（み）つかるといいね
    　👉 希望能找到想要它的人

  9. キーホルダーをつけて
    　👉 裝上鑰匙圈

  10. 位置（いち）がずらす
    　👉 把位置移開、錯開

  11. ラバーストラップはチカピョンだ
    　👉 橡膠吊飾是「チカピョン」（虛構角色名）

  12. クスリと笑（わら）う
    　👉 輕輕一笑、微微一笑

  13. 構（かま）いませんが、私（わたし）も愛着（あいちゃく）が湧（わ）いてきました
    　👉 雖然沒關係，不過我也開始有感情了

  14. 結果（けっか）はあまり芳（かんば）しくない
    　👉 結果不太理想（芳しく＝正面、常用於否定句）

  15. ていあらさんが徐（おもむろ）にスマホを弄（いじ）りだす
    　👉 ていあら小姐慢慢地開始操作手機

  16. スカートをはたきながら立（た）ち上（あ）がる
    　👉 拍掉裙子上的灰塵站起來

  17. パッチリ
    　👉 睜大眼睛（形容眼睛圓潤有神）

  18. 腕組（うでぐ）みをしながら見下（みお）ろす
    　👉 抱著手臂俯視

  19. チラリと見（み）る
    　👉 偷看一眼（與#25重複）

  20. 佳樹（よしき）がボソリとつぶやく
    　👉 佳樹喃喃低語

  21. 口（くち）から出（で）まかせに
    　👉 信口開河、胡說八道

  22. 野次馬（やじうま）根性（こんじょう）
    　👉 看熱鬧不嫌事大的心態

  23. スイーツバイキングのタダ券（けん）
    　👉 甜點吃到飽的免費券

  24. カドワラ
    　👉 可能是虛構地名或人物名（語境不明）

  25. 選挙（せんきょ）の件（けん）でゴチャついてる
    　👉 因為選舉的事變得很混亂

  26. 落（お）ち着（つ）かなくキョロキョロしてると、雑誌（ざっし）を覗（のぞ）き込（こ）んだまま八奈見（やなみ）さんがボソリとつぶやく
    　👉 我坐立不安地東張西望時，八奈見先生一邊盯著雜誌一邊低語。

  27. 首（くび）をかしげて見（み）せる
    　👉 歪著頭看（表示疑問、困惑或裝可愛）

  28. 面食（めんく）らう
    　👉 驚訝、嚇一跳（常指意料之外）

  29. 二人（ふたり）は犬猿（けんえん）の仲（なか）
    　👉 兩人是水火不容的關係（字面：像狗與猴子一樣不合）

  30. あれは単（たん）に絡（から）まれてただけだし
    　👉 那只是被找麻煩而已啦

91-117

  1. 額（ひたい）を押（お）さえ、溜息（ためいき）をつく
    　👉 按住額頭嘆氣

  2. お詫（わ）びしたい
    　👉 想道歉

  3. 途端（とたん）に人通（ひとどお）りが少（すく）なくなる
    　👉 突然行人變少了

  4. なんの念押（ねんお）しだ
    　👉 是什麼意思的再三叮囑？

  5. 画像（がぞう）コンプリート
    　👉 圖片收集完畢（用於轉蛋、收藏類遊戲中）

  6. 界隈（かいわい）は古（ふる）い商店街（しょうてんがい）
    　👉 這一帶是老式商店街

  7. 声（こえ）を軽（かる）く押（お）し潰（つぶ）してきたのは、恰幅（かっぷく）のいいおばさまの声（こえ）
    　👉 壓低聲音說話的是體格豐滿的大嬸

  8. 一瞬（いっしゅん）たじろいだていあらさんは、ペコリと額（ひたい）を下（さ）げる
    　👉 ていあら小姐一瞬間退縮了一下，然後鞠了一個躬

  9. みなまで言（い）うなとばかりにコクコク頷（うなず）く
    　👉 彷彿在說「不用再說了」，連連點頭

  10. バッジやトロフィー、垂（た）れ幕（まく）とか
    　👉 像是徽章、獎盃、懸掛布條等東西

  1. 招（まね）き入（い）れてくれた
    　👉 把我請進去了

  2. ぎこちなく座布団（ざぶとん）に正座（せいざ）する
    　👉 笨拙地在坐墊上跪坐

  3. ちょこんと座（すわ）り
    　👉 小小地坐著（形容安靜、可愛的姿態）

  4. 無言（むごん）のままモソモソと教科書（きょうかしょ）を取（と）り出（だ）す
    　👉 一語不發地慢吞吞拿出教科書

  5. 改善（かいぜん）の気配（けはい）がそこかしこに見（み）られるといいますか
    　👉 怎麼說呢，到處都能看到改善的跡象

  6. グッと拳（こぶし）を握（にぎ）り締（し）めるていあらさん
    　👉 ていあら小姐用力握緊拳頭

  7. ひっつめ髪（がみ）に太（ふと）い眉（まゆ）。首（くび）のホクロが白（しろ）い肌（はだ）にくっきりと浮（う）かんでいる
    　👉 把頭髮全部綁起來，濃眉。脖子上的痣在白皙的皮膚上很明顯。

  8. 怪訝（けげん）そうに首（くび）をかしげる
    　👉 露出困惑的表情歪頭

  9. モニョモニョとごまかすと、ていあらさんは楽（たの）しそうに身（み）を乗（の）り出（だ）す
    　👉 含糊其辭地敷衍後，ていあら小姐開心地湊近過來

  10. 所信表明（しょしんひょうめい）
    　👉 表明決心或方針（如政治宣言、個人表態）

  11. 言（い）いそびれました
    　👉 不小心沒說出口了（錯過機會）

  12. いざとなったら、温水（ぬくみず）さんを説得（せっとく）する取（と）って置（お）きがあったんですが、必要（ひつよう）ありませんでした
    　👉 真到必要的時候我本來打算說服溫水先生，不過結果用不上。

  13. 笑（わら）いながらごまかすていあらさん
    　👉 邊笑邊搪塞的ていあら小姐

  14. 俺（おれ）が身（み）を乗（の）り出（だ）すと、思（おも）わずたじろぐていあらさん
    　👉 我湊近時，ていあら小姐不禁退了一步

  15. 唇（くちびる）を小刻（こきざ）みに震（ふる）わせながら、上目遣（うわめづか）いに頷（うなず）く
    　👉 嘴唇微微顫抖，用上目視線點頭

  16. アクスタか何（なに）かだろか
    　👉 是立牌還是什麼的吧？（アクスタ＝アクリルスタンド）

  17. テンパリていあらさんが叫（さけ）んだ瞬間（しゅんかん）
    　👉 緊張混亂的ていあら小姐大喊出聲的瞬間
