# caoshaun.github.io 🚀

## 概要 📖
このリポジトリは、HexoとMateryテーマを使用して構築された個人ブログサイトのソースコードです。2020年に作成され、GitHub Pagesで公開されています。✨

## 特徴 ⭐
- **静的サイト生成:**  
  Hexoを使用して高速かつSEOに強い静的HTMLを生成します。⚡
- **洗練されたデザイン:**  
  Materyテーマに基づいたシンプルで美しいUIを実現。🎨
- **豊富な機能:**  
  - カテゴリ・タグ別記事一覧、アーカイブ機能 🗂️  
  - クライアントサイド検索（search.xml利用）🔍  
  - Live2Dキャラクターによるインタラクティブなユーザー体験 🤖

## サイト構成 🏗️
### ルートファイル 📂
- `index.html`: サイトのエントリーポイント 🏠
- `search.xml`: クライアントサイド検索用インデックス

### コンテンツ関連ディレクトリ 📝
- `about/`: 自己紹介ページ 👤
- `archives/`: 過去記事一覧 📚
- `categories/`: カテゴリ別一覧 🗃️
- `tags/`: タグ別一覧 🏷️
- `contact/`: お問い合わせページ 📞

### アセット管理 🎨
- `css/`: テーマおよびカスタムスタイルシート 🎀
- `js/`: ユーザーインタラクションおよびUIエフェクトを管理するJavaScript 🎛️
- `libs/`: 外部ライブラリ 📦
- `medias/`: 画像やアイコンなどのメディアファイル 🖼️

### Live2D関連 💡
- `live2d/` および `live2dw/`: Live2Dキャラクターの初期化やイベントハンドリング用スクリプト・アセット 🤩

## ビルド・デプロイ手順 🔧
1. **環境構築:**  
   - [Hexo](https://hexo.io/)をインストールし、依存プラグインを設定します。🛠️
2. **サイト生成:**  
   - コマンドラインで `hexo generate` を実行し、静的ファイルを生成。🖥️
3. **ローカル確認:**  
   - `hexo server` でローカルサーバーを起動し、動作確認。🔎
4. **デプロイ:**  
   - GitHub Pagesを利用して、生成されたファイルを自動的に公開。🌐

## カスタマイズ 🎨
- **設定ファイル:**  
  - ルートの `_config.yml` やテーマ内のテンプレートファイルでサイト全体の設定やレイアウトを調整。⚙️
- **スタイル・スクリプト:**  
  - `css/` と `js/` ディレクトリ内のファイルを編集して、デザインや動作のカスタマイズが可能。✏️
- **Live2D:**  
  - Live2Dキャラクターの設定は、`live2d/` および `live2dw/` 内のスクリプトで管理。🤖
